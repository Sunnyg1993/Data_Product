About the App

This App help users do some primary data checking, including:

  • what are the variables
  • types and length of each variables
  • plot of x ~ y
  • plot of x ~ y with color code (categorized?)

This App link http://rmarkdown.rstudio.com.

App interface

Use of this App

Step 1: upload your csv. file and check the data in “Table” and “Summary” tabs

Use of this App

Step 2: choose the variables you want to examine and plot (red ones, “Plot” tab); you can also add a category color code to your plot (green ones, “Plot + colorcode” tab)
